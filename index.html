Ruby, Twitter, BigQuery, Bootstrap

Nenshi-bot is a Twitter Bot created for the 2015 City of Calgary Hackathon.

Admiral Ackbar

Ruby, Sensu, HipChat, OpsGenie, JIRA, SQL, Java

Solium's Premier Montioring and Alerting Engine!

Anchored on sensu, Admiral Ackbar checks a variety of 'traps', ranging from SQL Queries, small java classes, http pings/curls, file system scans and everything in between - all against the rapdily evolving Shareworks platform.

These traps look at both business logic and platform stability, and provide actionable information and step-by-step resolution instructions.

Check out the presentations »

Though I haven't yet written my own plugin or handler, I've done some break-fix work for the 'opsgenie-heartbeat' plugin, and added some functionality to the 'opsgenie' handler.

opsgenie-heartbeat » opsgenie handler »

CTD Pipeline

Bamboo, rpmspec, EC2, yum

The 'Continuous Delivery Pipeline' built to provide rapid availablity of updates for Calgary Scientific's 'ResolutionMD' server software.

CTD starts with a custom Bamboo build stage that creates a 'product' rpm package. This package, along with the ResolutionMD server rpm packages are then pushed up to yum repositories hosted in Amazon's EC2. The new packages are then available for updating any installation of ResolutionMD.
